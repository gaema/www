# gaema.github.io

The gaema.ai landing page — a single self-contained static HTML page
(no build step, no dependencies), served by GitHub Pages at https://gaema.ai.

- `index.html` — the page (inline CSS).
- `CNAME` — custom apex domain (gaema.ai).
- `.nojekyll` — serve files as-is, no Jekyll processing.
